2023 Summer Internship Project

Project Description:

This project focuses on predicting the phosphate production of two key stations, Koch and Krupp, within the OCP Group company. Phosphate production is a crucial aspect of OCP's operations, and accurate forecasting can enhance efficiency and planning. The project employs a machine learning approach, particularly Long Short-Term Memory (LSTM) neural networks, to predict future phosphate production levels.
